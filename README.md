# Responsive Calculator

A simple, responsive calculator UI built with HTML, SCSS, and vanilla JavaScript.

## Features
- Responsive layout with a clean keypad grid
- Operator, action, and equals button styling
- SCSS source with compiled CSS

## Files
- `index.html` - Markup for the calculator layout
- `styles.scss` - SCSS source styles
- `styles.css` - Compiled CSS (link target in `index.html`)
- `script.js` - Calculator behavior

## Usage
1. Open `index.html` in a browser.
2. If you edit `styles.scss`, compile it to `styles.css`.

## Notes
- The UI uses a 4-column grid; the 0 and = keys span two columns for balance.
